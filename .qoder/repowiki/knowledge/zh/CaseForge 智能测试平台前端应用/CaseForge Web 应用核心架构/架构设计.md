- **入口与配置**：`main.ts` 初始化 Vue 应用，集成 Ant Design Vue（中文 locale）、Pinia 状态管理及自定义路由；`App.vue` 通过 `a-config-provider` 注入全局主题 `caseForgeTheme`。
- **路由与布局**：`router/index.ts` 定义 `/case-forge` 和 `/api-test` 双平台路由，均嵌套于 `PlatformLayout.vue`；路由守卫自动同步 URL 中的 `userName` 参数至本地存储。
- **状态管理**：`stores/caseForge.ts` 为核心业务 Store，管理项目列表、结构化文档、测试要点及案例生成队列等复杂状态，包含大量异步 Action 处理后端交互与轮询逻辑。
- **API 通信**：`api/client.ts` 封装 Axios 实例，通过请求拦截器自动注入 `X-User-Name` 头并设置动态 baseURL；提供涵盖项目管理、文档结构化、动态指令及场景库的完整 API 方法集。
- **工具与常量**：`utils/userContext.ts` 实现用户身份的持久化与同步；`constants/platform.ts` 定义平台元数据；`composables/useImmersiveWorkspace.ts` 提供沉浸式工作区的 UI 交互逻辑。
