- **分层架构**：遵循 NestJS 标准三层结构，`controller` 处理 HTTP 请求与 Swagger 文档，`service` 封装业务逻辑与事务控制，`entity` 定义 TypeORM 数据模型。
- **数据持久化**：基于 `TypeOrmModule` 集成，主实体为 `CaseProjectEntity`，并在服务层通过 `DataSource` 手动管理涉及 `CaseEditorEntity` 和 `StructDocEntity` 的级联删除事务。
- **模块导出**：在 `index.ts` 中显式导出 `ProjectManageService`，支持跨模块复用项目管理能力。
- **输入校验**：通过 `dto` 目录下的类结合 `class-validator` 实现请求参数的自动化校验。
